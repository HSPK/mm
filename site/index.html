Workflow

From folder to browsable library in four steps.

01

Index

Scan existing folders without changing original files.

02

Import

Copy or move new media into template-based destinations.

03

Enrich

Extract metadata, thumbnails, places, tags, and ratings.

04

Browse

Use the web UI, CLI, or REST API to explore the library.

Features

Small enough for a laptop. Structured enough for a server.

Safe by default

Read-only indexing, duplicate checks, and deterministic import paths.

Real database options

Start with a portable SQLite file, or register a PostgreSQL URL later.

Smart organization

Tags, albums, cameras, years, places, ratings, and generated collections.

Local intelligence

Optional OpenCLIP auto-tagging and offline reverse geocoding.

Bundled web UI

FastAPI serves the React app, thumbnails, streaming, auth, and APIs.

Scriptable CLI

Automate scanning, imports, config, cleanup, sync, and search.

Install

One package, no mandatory Docker stack.

Install litemm, initialize a library, and start the server. The web UI is bundled with the Python package. 

pipx install litemm
mm init ~/Photos
mm import ~/Downloads/Camera
mm server
