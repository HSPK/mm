Scan, import, tag, search, and browse photos, videos, and audio from one lightweight Python package.

Why MM

Built for lightweight, local media workflows.

Safe importing

Template-based imports, file hashing, duplicate detection, and read-only indexing keep your library predictable.

Smart discovery

Browse by tags, cameras, years, places, ratings, and generated smart albums.

Local intelligence

Optional OpenCLIP auto-tagging and offline reverse geocoding avoid mandatory cloud services.

One deployable package

FastAPI serves the bundled React UI, thumbnails, media streaming, auth, and library APIs from one Python process.

Install

Start with a local database, move to PostgreSQL later.

MM is distributed as litemm. The default SQLite setup is zero-config, while registered PostgreSQL URLs are supported for server-backed libraries. 

# Install
pipx install litemm

# Create or open a library
mm init ~/Photos

# Register PostgreSQL if needed
mm db add postgresql://user:pass@host:5432/mm
